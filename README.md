### Comment (배포 대비 설명)

- 어비스 리액터: '속성 부여시 해당 속성 저항 -10' 구현 불가로 사용자가 직접 감안 필요
- 뜨거운 열망의 증표: 적을 보스, 네임드 몬스터로 간주할 시 '아무대상 상변'을 최대치로 조정할 것
- 죽음을 부르는 관: '자신 석화'시 '공격 스택'을 최대치로 조정할 것
